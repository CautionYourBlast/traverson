Checklist for publishing a new release:
- bump version in package.json
- bump version in bower.json
- grunt
- commit & push to master
- npm publish
- create a new release on github
  (https://github.com/basti1302/traverson/releases/new)
  - Tag version: new version number, without any prefix or suffix
  - Release title === Tag version
  - empty description
  - add all four JS files from browser/dist as "binaries" to the release
  - Publish release
